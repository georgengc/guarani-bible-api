Guaraní Bible API (WORK IN PROGRESS, DOESN'T WORK YET)

Some simple search functions for all the different versions of the Bible in the Guaraní language.

First, you can get all of the different versions using getVersions()

[
    { "id": "GRN", "name": "Tupa Nandejara", "code": "GRN1913", "year": 1913 }
    { "id": "GPC", "name": "Nandejara Ne'e", "code": "GPC2006", "year": 2006 }
    { "id": "GPC", "name": "Nandejara Ne'e", "code": "GPC2006", "year": 2006 }
]

Then you can look at the books inside each version using getBooks('GCP') or getBooks('GRN') or getBooks('TTV')

[
    { "number": "1", "name": "mateo", "id": "mat", "chapters": 28 },
    { "number": "2", "name": "marcos", "id": "mar", "chapters": 16 },
    { "number": "3", "name": "lucas", "id": "luc", "chapters": 24 },
    { "number": "4", "name": "juan", "id": "jua", "chapters": 21 }
]

After that you can view a list of the chapters within a book using getChapters('GCP', 'mat')

[
    { number: 1, title: "Jesucristo ypy kuerakue" },
    { number: 2, title: "Karai arandu kuéra ou mitã andúvo" },
    { number: 3, title: "Juan Bautista oñemoñe'ẽ" },
    { number: 4, title: "Aña oha'ã vai ramo guare Jesúspe" },
    { number: 5, title: "Jesús oñemoñe'ẽ peteĩ yvyty ári guive" },
    ...
]

Finally you can get a specific verse by using getVerse('GCP', 'jua', 3, 16)

{ verse: "Tupã ohayhu etereígui opa yvypórape, ome'ẽ vaekue Ta'ýra peteĩete, 
                ani haguã omano umi hese ojeroviáva, oikove haguã uvei opa ára." }
